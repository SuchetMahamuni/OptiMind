# OptiMind

This repository contains the Flask backend for **OptiMind**, a behavior-driven study productivity application designed to help students stay consistent, avoid burnout, and improve focus.

---

## 🚀 Features

* User Authentication (JWT-based)
* Task Management
* Study Session Tracking
* Focus Score Calculation
* Daily Adaptive Goals
* Rule-based Intelligent Nudges

---

## 🧠 Tech Stack

* Python (Flask)
* SQLite (development)
* SQLAlchemy (ORM)

---

## 📂 Project Structure

* `routes/` → API endpoints
* `models/` → Database models
* `services/` → Business logic (rule engine, scoring)
* `utils/` → Helper functions

---

## ⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the server:

```bash
python app.py
```

---

## 🌐 Deployment

This backend is deployed using PythonAnywhere.
